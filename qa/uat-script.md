# UAT Script (現場5名)

## UAT-1（Week2）
- 目的: 現場導線の違和感抽出
- 参加: 寺務所スタッフ5名
- 時間: 2時間

### シナリオ
1. 世帯登録→故人追加
2. 行事作成→対象追加→出欠更新
3. 取引登録（お布施/塔婆料）
4. 取引フィルタで対象者確認
5. 監査確認（管理者のみ）

### 記録
- 1操作あたりクリック数
- 詰まり時間（秒）
- エラー文言理解可否

  - 主要フローの完了率 95%以上
  - 現場満足度（5段階）平均 4.0以上
